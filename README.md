# Praktyki
### Strona dla programu praktyk

### Copy `.env` file

```
cp .env.example .env
```

### Docker development

```
docker-compose up -d
```

### Installation

```
npm install
```

### Compiles and hot-reloads for development

```
npm run dev
```

### Compiles and minifies for production

```
npm run build
```

### Lints files

```
npm run lint
```

### Available containers

| service | container name | default external port |
| ------- |----------------| --------------------- |
| node    | praktyki-node  | 8080                  |
